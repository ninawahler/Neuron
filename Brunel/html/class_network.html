Detailed Description

class Network Here we have all the function and variables needed in order to run a simualation of a chosen number of neurons for a chosen time. Here we creat a vector containing all our neurons : the first part contains the exhibitories (differs by having a J=0.1 while Jinhi=-0.5) We also connect all neurons to each other randomly The class Network will be called in the main()
